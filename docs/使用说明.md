# 使用说明

## 客户与订单

- 一个客户可以添加多个平台身份和网名。
- 客户星级设为 1–5 星后，会自动出现在“星级 VIP”页面。
- 新建订单前需要先创建客户。
- 新建订单保存成功后，系统才会创建该客户的文件夹；复购订单只新增订单子文件夹。

## 推荐的订单流程

- 设计进度：无需设计、待设计、设计中、待确认、设计完成。
- 履约进度：待处理、待发货、已发货、已签收、已取消。
- 两条进度线可以独立修改。

## Excel 导入

1. 在“导入与导出”页面选择 `.xlsx`、`.xls` 或 `.ods` 文件。
2. 将表格列映射到客户名称、电话、微信、平台、平台网名、VIP、标签和备注。
3. 检查预览。缺少客户名称的行会标记为错误行。
4. 确认后，仅有效行写入数据库；重复手机号只提示，不自动合并。

## 数据恢复

- 恢复备份前，App 会自动保存当前数据库安全副本。
- 完整数据包用于迁移或长期归档；数据库备份用于恢复业务数据。
- 客户文件库中的大型设计文件不会每日重复完整备份，建议将文件库放在可靠磁盘，并定期执行完整数据导出。
